# Green 磁盘空间分析可视化工具

基于Python构建的磁盘空间分析可视化软件，可利用MFT主文件表进行分析，旨在将磁盘内各文件夹所占空间以直观的方式展示出来，帮助使用者更好地了解磁盘空间的使用情况。

## 一、文件结构

 `main.py`：程序入口文件  
 `scanner.py`：磁盘扫描主文件  
 `treemap.py`：treemap算法可视化展示文件  
 `models.py`：数据模型定义文件  
 `requirements.txt`：依赖说明  
 `run_visualizer.bat`：开发环境一键运行批处理文件  
 `build.bat`：一键打包脚本，将源码打包为独立 `.exe` 可执行文件  
 `build_icon_ico.py`：将`.png`格式的图标转换为`.ico`格式  
 `mft_scanner.py`：MFT读取和扫描文件  
 `icon.ico`和`icon.png`：图标文件  
  
## 二、运行环境

**操作系统**：Windows 7及更新版本（暂时未针对Linux和macOS进行适配）  

## 三、运行方式

### 方式一：直接运行 .exe

在项目的 Release 页面下载最新版本的 `GreenDiskVisualizer.exe`，双击即可运行.  
（仅amd64平台，问就是找不到那么多设备TAT...其余平台请按照**方式二**或**方式三**自行构建或运行源码）  
无需安装 Python 或其他软件。 

### 方式二：直接运行源码

需要 Python 3.8 及以上版本。

下载源码或克隆仓库后，
双击 `run_visualizer.bat`，或使用命令行：

```bash
cd 文件所在目录
python main.py
```

### 方式三：自行打包构建 .exe

arm64平台和x86平台需要自行从源码构建可执行文件：

1. 安装 Python 3.8或更新版本，下载源码；
2. 双击 `build.bat`，等待打包完成；
3. 生成的 `GreenDiskVisualizer.exe` 位于 `dist` 文件夹中；

该 `.exe` 文件可复制到同平台任何 Windows 7+ 电脑上直接运行，无需安装 Python。

## 四、主要功能

自动列出本机所有可用盘符。

  提供 **快速扫描** 和 **完整扫描** 两种模式：  
    -快速扫描：智能聚合模式，展示2层文件夹嵌套细节，兼顾速度和扫描质量；  
    -完整扫描：不限制深度，覆盖整个磁盘，展示5级文件夹嵌套细节；  
    （两种模式在管理员权限下运行时如能检测到MFT主文件表都会读取，可极大提升速度和扫描质量。）  
  显示扫描进度百分比、已扫描文件/目录数量，并估算剩余时间。  

## 五、注意事项

1.软件目前处于a测阶段，运行不稳定，不建议用于工作用途；  
2.目前版本如**未检测到MFT主文件表**将使用 Python标准库 扫描，**完整扫描大容量磁盘可能耗时较长**；  
3.本工具**无删除文件功能**，清理文件请自行定位删除。
